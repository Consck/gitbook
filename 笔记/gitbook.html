gitbook

gitbook安装过程(mac)

安装node

下载10.22.0版本pkg文件。否则容易出现版本问题引起的错误

安装完成后使用命令验证

$node -v
v10.22.0
$npm -v
6.14.6

安装gitbook

sudo npm install gitbook-cli -g

通过命令gitbook -V验证

CLI version: 2.3.0
GitBook version: 3.2.2

gitbook使用

  1. 创建文件夹，作为gitbook编写大本营
  2. 编写test.md文件,并将目录加入SUMMARY.md文件中
  3. 运行命令gitbook init
  4. 运行命令gitbook build
  5. 运行命令gitbook serve
  6. 通过http://localhost:4000查看
